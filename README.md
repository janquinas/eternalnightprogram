# Eternal Night - Program

A narrative-driven RPG project focused on atmosphere, choice, and immersive worldbuilding.

---

## About

Eternal Night is an original role-playing project designed to explore interactive storytelling through a web-based experience.
The project combines narrative design, system logic, and interface development to create a cohesive and evolving world.

This repository serves as both a development space and a documentation hub for the project.

---

## Vision

The goal of Eternal Night is to deliver a dark, engaging experience where player decisions shape the world and its outcomes.

The project emphasizes:

* Narrative depth
* Minimalist and immersive interface
* Modular and scalable code structure
* Continuous iteration and experimentation
